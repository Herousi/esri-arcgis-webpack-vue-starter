# my-starter

## Project setup
```
npm install
```

### Add Vue plugins
```
vue add vuetify
vue add electron-builder
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
